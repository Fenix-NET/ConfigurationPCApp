//Migration Commands for VisualStudio
1. Add new Migration : Add-Migration _MigrationName
2. Remove Last Migration : Remove-Migration
3. Execute Migration to update database: Update-Database

//Below command delete all table from database.
4. Clear all migration from database:Update-Database -Migration 0


5. Update database to specific migrations: Update-Database -Migration _MigrationName4
